# Scroll Spy React

Implémentation robuste d'un Scroll Spy avec React et IntersectionObserver.

## Technologies

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Installation

```sh
git clone <YOUR_GIT_URL>
cd scroll-spy
npm i
npm run dev
```

## Scripts

- `npm run dev` - Serveur de développement
- `npm run build` - Build de production
- `npm run preview` - Prévisualisation du build
- `npm run lint` - Vérification ESLint
